supporting hardware.

There is a demo of the emulator running at:

http://www.ubercomp.com/jslm32/src/

To run jslm32 on your own machine:

python -m SimpleHTTPServer

Open your browser:
http://localhost:8000/src

Click the Start! button.

Enjoy.
